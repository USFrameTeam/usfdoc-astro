下面介绍 USF 框架插件 的具体使用方法。所有操作都在游戏内完成，无需修改文件，通过命令、物品、界面进行。

## 一、基础命令

<Aside type="tip">
命令需要在聊天栏输入，前面加 `+` 号
</Aside>

| 命令 | 功能 | 示例 |
|------|------|------|
| `+cd` | 打开主菜单（核心功能入口） | `+cd` |
| `+land` | 创建领地（先选两个点，空手潜行+单击方块选点） | `+land` |
| `+unland` | 取消领地创建 | `+unland` |
| `+home` | 回家（返回设置的个人传送点） | `+home` |
| `+tpr` | 随机传送（需开启配置） | `+tpr` |
| `+tpaccept` | 接受别人发起的传送请求 | `+tpaccept` |
| `+back` | 返回死亡点或上一次传送前位置 | `+back` |
| `+op` | 打开管理员管理界面（OP 或 Owner 可用） | `+op` |
| `+die` | 自杀（如果配置中开启） | `+die` |
| `+unsleep` | 查看未睡觉的玩家列表 | `+unsleep` |
| `+usf` | 显示插件版本信息 | `+usf` |

<Aside type="caution">
部分命令需要在配置中开启对应功能才会生效。
</Aside>

## 三、核心功能详细用法

### 1. 领地系统

<Steps>

1. **选择第一个角点**：空手，潜行（按住 Shift）单击第一个角点（方块），会提示"已选定"

2. **选择对角点**：再单击对角点（不潜行），再次提示

3. **输入命令**：输入 `+land`，打开创建界面

4. **设置领地**：设置领地名、调整Y轴范围（可选预览），确认后扣除费用（需记分板货币）即可创建

</Steps>

**管理领地**：在主菜单 → 领地，点击已有领地，可编辑名称、欢迎语、成员权限、访客权限，添加/移除成员，删除领地等。

### 2. 传送点

<Tabs>
  <TabItem label="个人传送点">
    主菜单 → 传送系统 → 个人传送点 → 添加 → 设置名称、图标、位置（当前位置或已有坐标），可设为 Home
  </TabItem>
  <TabItem label="公共传送点">
    <Badge text="仅管理员" variant="caution" />
    同上，在公共传送点中添加
  </TabItem>
  <TabItem label="群组传送点">
    需先加入群组，在群组传送点中添加
  </TabItem>
  <TabItem label="玩家传送">
    主菜单 → 传送系统 → 传送玩家 → 选择目标玩家和方向（你过去 or 他过来），对方输入 `+tpaccept` 即可
  </TabItem>
</Tabs>

### 3. 群组系统

<Steps>

1. **创建群组**：主菜单 → 群组 → 新建群组 → 填写群名、公告

2. **加入群组**：主菜单 → 群组 → 加入群组 → 输入群ID → 等待群主同意

3. **群聊**：在聊天设置中切换到群聊模式，消息会发给所有群成员

4. **管理群组**：群主可加人、踢人、编辑公告、处理入群申请

</Steps>

### 4. 商店系统

**购买物品**：主菜单 → 商店 → 选择分组（如有）→ 点击商品 → 输入数量或拖动滑块 → 确认（消耗货币或物品）

**出售物品**：点击商品 → 输入要出售的数量 → 交出对应物品，获得货币

**管理员配置商店**：主菜单 → 管理界面 → 插件设置 → 全局商店设置 → 管理全局商店，可添加商品、设置货币、分组、限量等

### 5. 策略文件（自定义UI）

<Aside type="note">
策略文件物品 ID 为 `usf:config_file`，管理员可通过命令或创造模式获得
</Aside>

<Steps>

1. **获取策略文件**：管理员可通过命令或创造模式获得 `usf:config_file`

2. **编辑策略文件**：手持策略文件，打开主菜单 → 管理界面 → 编辑策略文件

3. **设置内容**：设置主界面标题、内容、二级菜单、按钮

4. **按钮类型**：执行命令、打开商店商品、传送点、子页面等

</Steps>

**使用方式**：
- 手持文件副手（或主手）右键空气打开界面
- 潜行右键方块可绑定该界面到方块（方块被右键时触发）
- 也可绑定到 NPC（右击 NPC 打开）

### 6. 小木斧（快速建造）

<Aside type="caution">
前提：配置中开启"小木斧"，手持木斧
</Aside>

<Steps>

1. **选择区域**：不潜行，左键/右键选择第一个方块，再点击另一个方块选择第二点

2. **打开界面**：潜行 + 右键打开小木斧界面

3. **执行操作**：可执行填充（全部/替换/轮廓）、导出为结构文件等操作

</Steps>

### 7. 悬浮字（文本展示）

**创建**：主菜单 → 管理界面 → 管理悬浮字 → 添加悬浮字 → 设置文本、坐标、备注

**修改**：点击已有悬浮字进行编辑

<Aside type="tip">
悬浮字本质是带有 Float 标签的蝙蝠，不会自然消失，可随意移动位置
</Aside>

## 四、特殊物品及标签用法

### 物品效果（自定义物品能力）

<Steps>

1. 使用命令或插件获取任意物品

2. 主菜单 → 管理界面 → 编辑物品特殊效果 → 选择物品栏位置 → 选择效果类型（击退、执行命令集、传送等）

3. 持有该物品攻击或使用时触发效果

</Steps>

### 锁定物品（防止丢弃/移动）

- 管理员在管理界面 → 插件设置 → 锁定物品设置中配置
- 配置指定槽位、物品ID、数量、限制标签
- 玩家若拥有对应标签，该槽位会被强制锁定为该物品（死亡不掉落，无法移动）

### 聊天头衔标签

给玩家添加标签 `usf.tag:头衔内容`，例如：

```
/tag 玩家 add usf.tag:§6[大佬]§r
```

玩家聊天时头衔会显示在名字前。使用 `usf.tag:Reset` 重置头衔。

## 五、管理员常用操作

### 打开管理界面

Owner 或 OP 输入 `+op` 或使用 `usf:op` 物品。

### 配置插件

管理界面 → 插件设置，可开关/调整几乎所有功能：

- 领地价格、大小限制
- 传送冷却、随机传送范围
- 商店货币、限量刷新
- 聊天格式、头衔默认值
- 日志记录内容
- 反作弊选项
- 游戏规则锁定
- 等等……

### 在线编辑公告

管理界面 → 公告设置 → 添加/编辑公告，支持多行文本、图标、顶置。

### 封禁/禁言玩家

管理界面 → 屏蔽/禁言玩家，选择玩家，设置禁言秒数或屏蔽聊天。

### 查看/删除领地

管理界面 → 管理玩家领地 或 删除领地（输入领地ID）。

### 全局事件配置

管理界面 → 配置全局事件，可设置在玩家进服、死亡、聊天、传送、维度切换、放置/破坏方块、攻击、睡觉时自动执行一组命令。

### 数据统计方案

管理界面 → 数据统计（原积分功能），为不同标签的玩家设置统计规则，自动增加记分板分数。

  </Card>

  </Card>

  </Card>

</Aside>

## 基础命令

| 命令 | 功能 |
|------|------|
| `+cd` | 打开主菜单 |
| `+op` | 打开管理界面（需要OP权限） |
| `+home` | 传送到Home点 |
| `+tpaccept` | 接受传送请求 |
| `+back` | 返回死亡点/上次传送位置 |
| `+tpr` | 随机传送 |
| `+land` | 创建领地 |
| `+unland` | 取消领地创建 |
| `+die` | 自杀 |
| `+unsleep` | 查看未睡觉玩家 |
| `+usf` | 显示插件版本 |

## 初始化

<Aside type="danger">
首次使用必须初始化！
</Aside>

- 单人存档：在聊天栏输入 `/function get_owner`
- 服务器：在后台输入 `/scriptevent usf:get_owner 玩家名`

日志系统可以记录服务器里发生的所有事情，方便管理员查案！

## 日志能记录什么？

| 记录类型 | 说明 |
|----------|------|
| 聊天记录 | 谁说了什么 |
| 进出记录 | 谁进来了、谁出去了 |
| 方块记录 | 谁挖了什么、放了什么 |
| 死亡记录 | 谁死了、被谁杀死的 |
| 容器记录 | 谁开了箱子、拿了什么东西 |
| 传送记录 | 谁传送到哪里 |
| 告示牌记录 | 谁改了告示牌内容 |

## 为什么需要日志？

- **查熊孩子**：谁把你的房子拆了？
- **找回物品**：谁拿了你箱子里的东西？
- **解决纠纷**：有证据就好说话！

## 日志在哪里看？

日志需要管理员设置「日志服务器」，记录会发送到那个服务器。

管理员可以在管理界面 → 插件设置 → 日志设置 里开启和配置。

## 我能看到日志吗？

普通玩家看不到日志，只有管理员（OP）才能查看。

如果你怀疑有人搞破坏，可以找管理员帮你查！

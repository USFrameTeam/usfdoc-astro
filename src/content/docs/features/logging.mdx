日志系统可以记录服务器各种事件，并发送到外部日志服务器。

## 日志类型

| 日志代码 | 说明 |
|----------|------|
| `chat` | 聊天记录 |
| `jl` | 进出游戏记录 |
| `bb` | 破坏方块记录 |
| `pb` | 放置方块记录 |
| `ib` | 与方块交互记录 |
| `die` | 死亡记录 |
| `kill` | 击杀记录 |
| `chest` | 容器操作记录 |
| `tp` | 传送记录 |
| `di` | 维度切换记录 |
| `sign` | 告示牌更改记录 |
| `lo` | 玩家位置记录（每60秒） |

## 配置日志服务器

<Steps>

1. 部署日志服务器（需要支持 HTTP 接收）

2. 在管理界面 → 插件设置 → 日志设置

3. 开启日志功能

4. 填写服务器地址

5. 选择要记录的日志类型

</Steps>

## 日志配置

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| `log.able` | 启用日志 | true |
| `log.address` | 日志服务器地址 | `http://127.0.0.1:1024/` |
| `log.down` | 重连间隔（秒） | 30 |
| `log.allow` | 启用的日志类型 | [] |

## 日志格式

日志以 HTTP 请求发送，包含以下信息：
- 事件类型
- 事件详情
- 玩家信息
- 时间戳
- 位置信息

## 容器记录

记录容器的打开和物品变动：
- 哪些玩家打开了容器
- 物品的增减情况
- 容器类型和位置

## 告示牌记录

记录告示牌内容的修改：
- 修改前的内容
- 修改后的内容
- 修改位置

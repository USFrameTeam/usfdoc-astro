群组系统让玩家可以组队，方便一起玩、一起建、一起浪！

## 服主看这里

### 开启群组功能

管理界面 → 插件设置 → 群组设置

| 配置 | 说明 | 建议 |
|------|------|------|
| 启用群组 | 开关 | 建议开 |
| 最大群组数 | 每人能建几个 | 默认2，够用了 |

## 玩家看这里

### 创建群组

<Steps>

1. 主菜单 → 群组 → 新建群组

2. 给群组取个响亮的名字（比如"肝帝联盟"、"建筑狂魔"）

3. 写个公告（可选，比如"每晚8点集合"）

4. 创建成功！你会得到一个群组ID

</Steps>

<Aside type="note">
每人最多只能创建 2 个群组，别贪心~
</Aside>

### 加入群组

<Steps>

1. 让群主把群组ID告诉你

2. 主菜单 → 群组 → 加入群组

3. 输入群组ID

4. 等群主同意（群主可能在肝，耐心等等）

</Steps>

### 群组功能

#### 群聊
切换到群聊模式，你说的话只有群组里的人能看到，再也不怕被偷听！

#### 群组传送点
群组成员可以创建共享传送点，方便集合，再也不用跑半天找队友！

#### 群组公告
群主可以发布公告，比如"今晚8点集合打龙"、"谁偷了我的钻石？"。

### 管理群组

群主可以：
- 同意/拒绝入群申请
- 踢出成员（不听话的踢了）
- 编辑公告
- 解散群组（散伙饭记得吃）

### 退出群组

主菜单 → 群组 → 选择群组 → 退出群组

想走就走，没人拦你~

群组系统让你可以和朋友组队，方便一起玩！

## 怎么创建群组？

<Steps>

1. 主菜单 → 群组 → 新建群组

2. 给群组取个名字

3. 写个公告（可选）

4. 创建成功！你会得到一个群组ID

</Steps>

<Aside type="note">
每人最多只能创建 2 个群组哦！
</Aside>

## 怎么加入别人的群组？

<Steps>

1. 让群主把群组ID告诉你

2. 主菜单 → 群组 → 加入群组

3. 输入群组ID

4. 等群主同意

</Steps>

## 群组里能干什么？

### 群聊
切换到群聊模式，你说的话只有群组里的人能看到。

### 群组传送点
群组成员可以创建共享的传送点，方便集合！

### 群组公告
群主可以发布公告，所有成员都能看到。

## 怎么管理群组？

群主可以：
- 同意/拒绝入群申请
- 踢出成员
- 编辑公告
- 解散群组

## 我想退出群组

主菜单 → 群组 → 选择群组 → 退出群组

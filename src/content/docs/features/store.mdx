商店系统让服务器有经济循环，玩家可以买卖物品，再也不用到处找人交易了！再也不用体验"挖矿三小时，钻石零颗"的痛苦了！

## 服主看这里

### 开启商店功能

管理界面 → 插件设置 → 全局商店设置

| 功能 | 说明 | 建议 |
|------|------|------|
| 启用商店 | 开关 | 必须开！ |
| 货币记分板 | 用什么货币 | 必须设置！ |

### 添加商品

管理界面 → 全局商店管理 → 添加商品

商品类型：
- **售卖物品**：玩家花钱买物品（最常用）
- **收购物品**：玩家卖物品换钱（回收垃圾）
- **礼品**：免费领取（可限量，发福利用）

### 商品配置

| 配置 | 说明 | 建议 |
|------|------|------|
| 标题 | 商品显示名称 | 简洁明了，别太长 |
| 价格 | 多少钱 | 别太贵，也别太便宜 |
| 币种 | 用什么货币 | 必须设置 |
| 分组 | 商品分类 | 建议分组，方便查找 |
| 全局限量 | 服务器总限量 | 0为不限 |
| 个人限购 | 每人限量 | 0为不限 |
| 刷新方式 | 限量刷新 | 看需求 |

### 分组管理

管理界面 → 全局商店管理 → 配置分组

建议分组：
- 建筑材料（砖头、木头、石头）
- 食物药水（回血、加buff）
- 工具武器（锄头、剑、弓）
- 稀有物品（钻石、下界合金）

## 玩家看这里

### 怎么打开商店？

主菜单 → 商店

### 买东西

<Steps>

1. 打开商店，浏览商品

2. 点击你想买的商品

3. 输入购买数量（或者用滑动条）

4. 确认购买（扣游戏币）

</Steps>

### 卖东西

<Steps>

1. 打开商店，找到可以「出售」的商品

2. 点击商品

3. 输入你要卖的数量

4. 把对应物品交出去，拿到游戏币！

</Steps>

### 以物易物

除了用游戏币，还可以用物品换物品！

比如：用 64 个泥土换 1 个钻石，童叟无欺！再也不用体验"No Diamonds?"了！

### 商品限量

有些商品有限制：
- **全局限量**：整个服务器只能买这么多，先到先得！
- **个人限购**：每个人只能买这么多，防止囤货

### 选择数量的方式

商品可能有不同的数量选择方式：
- 手动输入数字（精确控制）
- 滑动条选择（1-10、1-16、1-64、1-256）
- 快速购买（点击直接买1个）

## 我的游戏币在哪看？

游戏币通常是记分板上的分数，管理员会告诉你用哪个记分板。

输入 `+cd` 打开主菜单，有些地方会显示你的余额。

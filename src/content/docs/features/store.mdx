商店系统让你可以和其他玩家买卖物品！

## 怎么打开商店？

主菜单 → 商店

## 怎么买东西？

<Steps>

1. 打开商店

2. 浏览商品（可能有分类标签）

3. 点击你想买的商品

4. 输入购买数量

5. 确认购买（会扣你的游戏币）

</Steps>

## 怎么卖东西？

<Steps>

1. 打开商店

2. 找到可以「出售」的商品

3. 点击商品

4. 输入你要卖的数量

5. 把对应物品交出去，就能拿到游戏币了！

</Steps>

## 商店里有什么？

商店里的商品由管理员设置，可能包括：
- 各种方块和材料
- 食物和药水
- 工具和武器
- 甚至可以执行命令！

## 商品限量吗？

有些商品可能有限制：
- **全局限量**：整个服务器只能买这么多
- **个人限购**：每个人只能买这么多

限量商品会定期刷新！

## 我的游戏币在哪看？

游戏币通常是记分板上的分数，管理员会告诉你用哪个记分板。

商店系统允许服务器创建全局商店，玩家可以购买和出售物品。

## 商店功能

### 购买物品

<Steps>

1. 主菜单 → 商店

2. 选择商品分组（如有）

3. 点击要购买的商品

4. 输入数量或使用滑块

5. 确认购买（消耗货币）

</Steps>

### 出售物品

<Steps>

1. 主菜单 → 商店

2. 选择可出售的商品

3. 输入要出售的数量

4. 交出对应物品，获得货币

</Steps>

## 商品类型

| 类型 | 说明 |
|------|------|
| 购买类 | 玩家用货币购买物品 |
| 出售类 | 玩家出售物品获得货币 |
| 命令类 | 执行指定命令 |

## 商品配置

| 配置项 | 说明 |
|--------|------|
| `item` | 物品ID（如 `minecraft:diamond`） |
| `price` | 价格 |
| `money` | 货币记分板ID |
| `count` | 每次购买数量 |
| `group` | 商品分组 |
| `title` | 商品显示名称 |
| `description` | 商品描述 |
| `icon` | 商品图标 |
| `global_count` | 全局限量（0为不限） |
| `personal_count` | 个人限购（0为不限） |
| `update_type` | 刷新类型 |
| `update_time` | 刷新时间（秒） |

## 管理员配置商店

<Steps>

1. 输入 `+op` 打开管理界面

2. 插件设置 → 全局商店设置

3. 管理全局商店 → 添加/编辑商品

4. 设置商品信息、货币、分组等

</Steps>

## 商店配置

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| `store.able` | 启用商店 | false |
| `store.moneys` | 货币记分板ID | - |
| `store.groups` | 商品分组配置 | {} |

## 限量商品

支持两种限量方式：
- **全局限量**：所有玩家共享购买次数
- **个人限购**：每个玩家独立的购买次数

可设置刷新时间，定期重置购买次数。

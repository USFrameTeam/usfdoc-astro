商店系统让服务器有经济循环，玩家可以买卖物品，再也不用到处找人交易了！再也不用体验"挖矿三小时，钻石零颗"的痛苦了！

## 服主指南

### 功能配置

管理界面 → 插件设置 → 全局商店设置

| 功能 | 说明 | 建议 |
|------|------|------|
| 启用商店 | 功能开关 | 推荐开启 |
| 货币记分板 | 用于交易结算的记分板 | 必须配置 |

### 商品管理

管理界面 → 全局商店管理 → 添加商品

商品类型：
- **售卖商品**：玩家支付货币获取物品（主流交易方式）
- **收购商品**：玩家出售物品获取货币（资源回收渠道）
- **礼品商品**：免费领取，可设置限量（活动福利发放）

### 商品配置

| 配置项 | 说明 | 建议 |
|--------|------|------|
| 标题 | 商品显示名称 | 简洁明了，突出卖点 |
| 价格 | 交易价格 | 合理定价，平衡经济 |
| 币种 | 交易使用的货币 | 必须指定 |
| 分组 | 商品分类归属 | 建议分类，便于管理 |
| 全局限量 | 服务器总供应量 | 0表示不限量 |
| 个人限购 | 单人购买上限 | 0表示不限购 |
| 刷新方式 | 限量商品的补货周期 | 按需设置 |

### 分组管理

管理界面 → 全局商店管理 → 配置分组

建议分类：
- 建筑材料（砖石、木材、石材）
- 食物药水（回复类、增益类）
- 工具武器（农具、兵器、弓弩）
- 稀有物品（钻石、下界合金）

## 玩家指南

### 怎么打开商店？

主菜单 → 商店

### 买东西

你挖矿挖了半天，结果挖到的都是石头？或者你想买点食物回血？商店系统就是为了解决这个问题！

<Steps>

1. 打开商店，浏览商品

2. 点击你想买的商品

3. 输入购买数量（或者用滑动条）

4. 确认购买（扣游戏币）

</Steps>

### 卖东西

你挖矿挖了一堆没用的东西？比如圆石、泥土？商店系统支持收购，你可以卖掉这些垃圾换钱！

<Steps>

1. 打开商店，找到可以「出售」的商品

2. 点击商品

3. 输入你要卖的数量

4. 把对应物品交出去，拿到游戏币！

</Steps>

### 以物易物

除了用游戏币，还可以用物品换物品！

比如：用 64 个泥土换 1 个钻石，童叟无欺！再也不用体验"No Diamonds?"了！

### 商品限量

有些商品有限制：
- **全局限量**：整个服务器只能买这么多，先到先得！
- **个人限购**：每个人只能买这么多，防止囤货

### 选择数量的方式

商品可能有不同的数量选择方式：
- 手动输入数字（精确控制）
- 滑动条选择（1-10、1-16、1-64、1-256）
- 快速购买（点击直接买1个）

## 我的游戏币在哪看？

游戏币通常是记分板上的分数，管理员会告诉你用哪个记分板。

输入 `+cd` 打开主菜单，有些地方会显示你的余额。

聊天系统让玩家交流更方便，支持私聊、群聊、头衔等功能。再也不用体验"苦力怕把你家炸了，你却不知道是谁干的"了！

## 服主看这里

### 聊天格式

管理界面 → 插件设置 → 聊天设置

| 配置 | 说明 | 建议 |
|------|------|------|
| 聊天格式 | 聊天显示样式 | 可自定义 |
| 消息长度 | 最多能说多少字 | 默认1024够了 |
| 禁用聊天 | 关闭聊天 | 慎用！玩家会跑光 |
| 清空聊天栏 | 自动清屏 | 看需求 |

### 设置头衔

管理员给玩家添加标签：
```
/tag 玩家名 add usf.tag:§6[大佬]§r
```

取消头衔：
```
/tag 玩家名 add usf.tag:Reset
```

<Aside type="tip">
头衔支持颜色代码，可以做出五颜六色的效果！比如 `§6` 是金色，`§c` 是红色。
</Aside>

## 玩家看这里

### 聊天模式

主菜单 → 聊天设置，可以切换聊天模式：

| 模式 | 说明 |
|------|------|
| 公共聊天 | 所有人都能看到你说的话 |
| 私聊 | 只有你选的那个人能看到 |
| 群聊 | 只有你群组里的人能看到 |

### 私聊

<Steps>

1. 主菜单 → 聊天设置

2. 切换到「私聊」模式

3. 选择你想私聊的玩家

4. 之后你说的话只有他能看到

</Steps>

<Aside type="tip">
私聊适合说悄悄话，比如"我发现钻石矿了，别告诉别人"、"苦力怕在你家旁边"。
</Aside>

### 聊天头衔

管理员可以给你设置「头衔」，显示在名字前面。

比如：`[大佬] 玩家名 >> 大家好！`

想换头衔？找管理员申请！

### 我被禁言了怎么办？

别担心，禁言会自动解除的！时间到了就能说话了。

如果你觉得是误禁，可以找管理员申诉。

### 留言板

主菜单 → 留言板

你可以：
- 看别人留的言
- 自己留言（比如"求组队"、"出售钻石"）
- 删除自己的留言

<Aside type="tip">
留言板是个好东西，可以发交易信息、找队友、或者单纯吐槽。
</Aside>

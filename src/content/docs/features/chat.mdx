聊天系统让你可以和服务器里的朋友交流！

## 聊天模式

主菜单 → 聊天设置，可以切换聊天模式：

| 模式 | 说明 |
|------|------|
| 公共聊天 | 所有人都能看到你说的话 |
| 私聊 | 只有你选的那个人能看到 |
| 群聊 | 只有你群组里的人能看到 |

## 我想和某个人私聊

<Steps>

1. 主菜单 → 聊天设置

2. 切换到「私聊」模式

3. 选择你想私聊的玩家

4. 之后你说的话只有他能看到

</Steps>

## 聊天头衔是什么？

管理员可以给玩家设置「头衔」，显示在名字前面。

比如：`[大佬] 玩家名 >> 大家好！`

### 怎么设置头衔？

管理员输入：
```
/tag 玩家名 add usf.tag:§6[大佬]§r
```

### 怎么取消头衔？

管理员输入：
```
/tag 玩家名 add usf.tag:Reset
```

## 我被禁言了怎么办？

别担心，禁言会自动解除的！时间到了就能说话了。

如果你觉得是误禁，可以找管理员帮忙解除。

## 留言板

主菜单 → 留言板

你可以：
- 看别人留的言
- 自己留言
- 删除自己的留言

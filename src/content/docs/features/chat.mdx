聊天系统提供格式化聊天、私聊、群聊等功能。

## 聊天模式

| 模式 | 说明 |
|------|------|
| 公共聊天 | 所有人可见 |
| 私聊 | 只有目标玩家可见 |
| 群聊 | 群组成员可见 |

切换方式：主菜单 → 聊天设置

## 聊天格式

默认格式：`[/tag]/sender >> /text`

### 可用变量

| 变量 | 说明 |
|------|------|
| `/sender` | 发送者名称 |
| `/tag` | 聊天头衔 |
| `/text` | 聊天内容 |

## 聊天头衔

### 设置头衔

管理员给玩家添加标签：
```
/tag 玩家名 add usf.tag:§6[大佬]§r
```

### 重置头衔

```
/tag 玩家名 add usf.tag:Reset
```

## 聊天配置

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| `chat.format` | 聊天格式 | `[/tag]/sender >> /text` |
| `chat.length` | 消息长度限制 | 1024 |
| `chat.disable` | 禁用聊天 | false |
| `chat.clear` | 清空聊天栏 | false |

## 留言板

主菜单 → 留言板

玩家可以：
- 查看留言
- 发布留言
- 清空自己的留言

## 禁言系统

管理员可以禁言玩家：
- 主菜单 → 管理界面 → 屏蔽/禁言玩家
- 设置禁言时间（秒）
- 禁言期间玩家无法发送消息

传送系统是 USF 的核心功能之一，提供多种传送方式。

## 个人传送点

每个玩家可以创建最多 50 个个人传送点，并可设置一个 Home 点。

### 创建传送点

<Steps>

1. 主菜单 → 传送系统 → 个人传送点

2. 点击「添加」创建新传送点

3. 设置名称、图标、位置（当前位置或手动输入坐标）

4. 可勾选「设为 Home」作为快速回家点

</Steps>

### 使用 Home 点

- 命令：`+home`
- 功能：立即传送到设置的 Home 点

## 玩家互传 (TPA)

<Steps>

1. 主菜单 → 传送系统 → 传送玩家

2. 选择目标玩家

3. 选择传送方向：
   - 「我过去」- 传送到对方位置
   - 「他过来」- 对方传送到你位置

4. 对方收到请求后输入 `+tpaccept` 接受

</Steps>

<Aside type="note">
传送请求有冷却时间（默认30秒），可在配置中修改。
</Aside>

## 随机传送

命令：`+tpr`

将玩家传送到随机位置，范围可在配置中设置（默认 10000 格）。

## 返回死亡点

命令：`+back`

返回上一次死亡的位置或上次传送前的位置（需开启 `tp.back`）。

## 传送配置

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| `tp.per` | 启用个人传送点 | true |
| `tp.per_count` | 个人传送点上限 | 50 |
| `tp.back` | 启用返回死亡点 | false |
| `tp.die` | 死亡后自动传送回死亡点 | false |
| `tp.down` | 传送冷却时间（秒） | 30 |
| `tp.random_range` | 随机传送范围 | 10000 |
| `tp.random_end` | 允许传送到末地 | true |
| `tp.animation` | 传送动画 | false |

## 公共传送点

<Aside type="caution">
仅管理员可创建公共传送点
</Aside>

管理员可在管理界面创建公共传送点，所有玩家都可使用。

## 群组传送点

需先加入群组，群组成员可在群组内创建共享传送点。

传送系统让你在服务器里快速移动，不用跑断腿！

## 怎么打开传送菜单？

在聊天栏输入 `+cd` 打开主菜单，点击「传送系统」按钮。

## 我想回家！

<Steps>

1. 先设置一个家：主菜单 → 传送系统 → 个人传送点 → 添加传送点

2. 给传送点取个名字（比如"我的家"）

3. 勾选「设为Home」

4. 以后想回家，直接输入 `+home` 就行了！

</Steps>

<Aside type="tip">
你最多可以创建 50 个传送点，但只能有 1 个 Home 点哦！
</Aside>

## 我想传送到朋友那里

<Steps>

1. 主菜单 → 传送系统 → 传送玩家

2. 选择你想传送到的朋友

3. 选择「我过去」（或者让他传送到你，选「他过来」）

4. 朋友会收到一个请求，他需要输入 `+tpaccept` 同意

</Steps>

<Aside type="note">
传送有冷却时间（默认30秒），不能一直传传传！
</Aside>

## 随机传送

输入 `+tpr` 可以把你传送到一个随机位置，适合探险！

## 我死了，想回到死亡地点

输入 `+back` 可以回到你上次死亡的地方（需要管理员开启这个功能）。

## 我被传送走了，想回到原来的地方

输入 `+back` 可以回到你上次传送前的位置。

## 传送点太多找不到？

传送点会按维度分类显示：
- 🌍 主世界
- 🔥 下界
- 🌌 末地

每个传送点都可以设置一个图标，方便你辨认！

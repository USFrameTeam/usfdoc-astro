传送系统让玩家在服务器里快速移动，再也不用跑半小时去找队友了！

## 服主看这里

### 开启传送功能

管理界面 → 插件设置 → 传送设置

| 功能 | 说明 | 建议 |
|------|------|------|
| 个人传送点 | 玩家自己设置传送点 | 必开，不开玩家会骂你 |
| 传送点上限 | 每人能设几个 | 默认50，够用了 |
| 玩家互传 | TPA功能 | 建议开，方便组队 |
| 返回死亡点 | 死后回去捡装备 | 新手服建议开，不然哭死 |
| 冷却时间 | 传送间隔 | 30秒，别太长 |
| 随机传送 | 开盲盒式传送 | 看地图大小决定范围 |
| 传送动画 | 酷炫的摄像机效果 | 想装逼就开 |

### 公共传送点

管理界面 → 传送设置 → 世界公共点

设置一些常用地点（出生点、商店区、活动区），玩家可以直接传送，省得跑断腿。

## 玩家看这里

### 我想回家！

<Steps>

1. 主菜单 → 传送系统 → 个人传送点 → 添加传送点

2. 给传送点取个名字（比如"温馨小窝"、"秘密基地"）

3. 勾选「设为Home」

4. 以后想回家，直接输入 `+home` 就行了！

</Steps>

<Aside type="tip">
你最多可以创建 50 个传送点，但只能有 1 个 Home 点！别贪心~
</Aside>

### 我要找朋友！

<Steps>

1. 主菜单 → 传送系统 → 传送玩家

2. 选择你想找的朋友

3. 选择「我过去」或者「他过来」

4. 朋友输入 `+tpaccept` 同意

</Steps>

<Aside type="note">
传送有冷却时间（默认30秒），不能当瞬移机器用！
</Aside>

### 我要探险！

输入 `+tpr` 随机传送，去一个完全陌生的地方，开盲盒式探险！

运气好可能传到钻石矿，运气不好可能传到岩浆里，谨慎使用！

### 我死了想回去捡装备！

输入 `+back` 回到死亡地点，赶紧去捡你的钻石装备！

<Aside type="danger">
如果管理员没开这个功能，你就只能跑回去了，哭~
</Aside>

### 我想分享传送点给朋友

<Steps>

1. 选择一个传送点

2. 点击「分享」按钮

3. 设置分享时间（10秒 ~ 10分钟）

4. 选择分享给「所有玩家」还是「部分玩家」

</Steps>

<Aside type="tip">
分享的传送点是临时的，时间到了就没了，抓紧时间！
</Aside>

## 传送动画

管理员可以开启「传送动画」，传送时会有酷炫的摄像机效果：
- 镜头升起（感觉自己要飞了）
- 移动到目标位置上方（鸟瞰大地）
- 缓缓降落（优雅落地）

不开动画的话就是瞬间传送，看你喜欢哪种风格！

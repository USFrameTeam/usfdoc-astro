领地系统可以保护你的房子和建筑，防止被熊孩子破坏！

## 怎么创建领地？

<Steps>

1. **选第一个角**：空手，按住 Shift（潜行），点击一个方块

2. **选对角**：松开 Shift，点击对角位置的方块

3. **打开创建界面**：输入 `+land`

4. **设置领地**：
   - 给领地取个名字
   - 调整高度范围（默认是从选的两个点的高度）
   - 确认创建（可能需要花一些游戏币）

</Steps>

<Aside type="tip">
选点的时候会有粒子效果显示你选的区域，看清楚再确认！
</Aside>

## 怎么管理我的领地？

主菜单 → 领地 → 点击你要管理的领地

你可以：
- 改领地名字
- 设置欢迎语（别人进入时会看到）
- 添加成员（让朋友可以在你的领地里建造）
- 设置权限（谁能挖、谁能放方块）

## 领地权限是什么？

| 权限 | 说明 |
|------|------|
| 破坏方块 | 能不能挖东西 |
| 放置方块 | 能不能放东西 |
| 与方块交互 | 能不能开箱子、用工作台 |
| 与实体交互 | 能不能和村民交易 |

## 我不想玩了，怎么删除领地？

主菜单 → 领地 → 选择领地 → 删除领地

## 常见问题

### 为什么我创建不了领地？

- 检查管理员有没有开启领地功能
- 确认你选了两个点（一个角和对角）
- 检查你有没有足够的游戏币

### 为什么别人在我的领地还能挖东西？

- 确认你创建的是「三维领地」（有高度限制）
- 检查访客权限设置

### 我能创建几个领地？

默认每人最多 2 个领地，管理员可以调整。

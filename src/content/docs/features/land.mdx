领地系统允许玩家保护自己的建筑区域。

## 创建领地

<Steps>

1. **选择第一个角点**：空手，潜行（按住 Shift）单击方块

2. **选择对角点**：不潜行，单击对角位置的方块

3. **打开创建界面**：输入 `+land` 或主菜单 → 领地 → 创建

4. **设置领地信息**：
   - 领地名称
   - Y轴范围（高度限制）
   - 预览领地区域

5. **确认创建**：扣除相应费用（需记分板货币）

</Steps>

<Aside type="tip">
选点时会显示粒子效果，帮助你确认领地区域。
</Aside>

## 领地权限

### 权限类型

| 权限代码 | 说明 |
|----------|------|
| `bb` | 破坏方块 |
| `pb` | 放置方块 |
| `ib` | 与方块交互（如打开箱子） |
| `ie` | 与实体交互 |

### 权限层级

| 层级 | 说明 |
|------|------|
| 管理员 (4) | 服务器 OP，拥有所有权限 |
| 领地主 (3) | 领地创建者 |
| 领地成员 (2) | 被添加为成员的玩家 |
| 群组成员 (1) | 领地主所在群组的成员 |
| 访客 (0) | 其他所有玩家 |

## 管理领地

主菜单 → 领地 → 选择已有领地

可进行以下操作：
- 编辑领地名称
- 设置欢迎语
- 添加/移除成员
- 设置成员权限
- 设置访客权限
- 删除领地

## 领地配置

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| `land.able` | 启用领地系统 | false |
| `land.max` | 每人最大领地数 | 2 |
| `land.board` | 领地费用记分板ID | - |
| `land.price` | 创建费用 | 0 |
| `land.must` | 强制扣除费用 | false |
| `land.show` | 进入领地提示文本 | `§e你已进入/name的领地！` |
| `land.mode` | 领地内游戏模式切换 | false |

## 领地提示

进入领地时会显示：
- 领地名称
- 领地主名称
- 当前权限层级
- 欢迎语（如果设置）

## 常见问题

### 无法创建领地？

- 检查领地系统是否开启（`land.able`）
- 确认已选择两个对角点
- 检查是否有足够货币

### 领地内可以破坏方块？

- 确认圈地为三维模式
- 检查访客权限设置

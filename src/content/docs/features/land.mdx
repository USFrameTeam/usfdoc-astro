领地系统是服务器防熊孩子的核心功能，没有之一！再也不用担心"苦力怕把你家炸了"了！

## 服主看这里

### 开启领地功能

管理界面 → 插件设置 → 领地设置

| 功能 | 说明 | 建议 |
|------|------|------|
| 启用领地 | 开关 | 必须开！ |
| 最大领地数 | 每人能建几个 | 建议3-5，太少了玩家会抱怨 |
| 费用记分板 | 用什么货币 | 必须设置，不然会报错！ |
| 创建费用 | 建领地要多少钱 | 建议设点费用，防止滥用 |
| 领地模式 | 进入领地自动切冒险模式 | 建议开，防破坏 |

<Aside type="danger">
领地功能必须设置费用记分板，否则创建领地会报错！这是最常见的错误！
</Aside>

## 玩家看这里

### 创建领地

你辛辛苦苦盖了个房子，结果出门挖矿回来发现被熊孩子拆了？或者被苦力怕炸了个洞？领地系统就是为了解决这个问题！

<Steps>

1. **选第一个角**：空手，按住 Shift（潜行），点击一个方块
   - 会看到粒子效果，说明选中了

2. **选对角**：松开 Shift，点击对角位置的方块
   - 这时候会看到一个框框，那就是你的领地范围

3. **打开创建界面**：输入 `+land`

4. **设置领地**：
   - 给领地取个名字，比如"我的钻石屋"、"苦力怕禁区"、"别动我的箱子"
   - 调整高度范围（默认是从选的两个点的高度）
   - 确认创建（扣游戏币）

</Steps>

<Aside type="tip">
选点的时候会有粒子效果，看清楚再确认！选错了可以取消重来。
</Aside>

### 管理领地

主菜单 → 领地 → 点击你要管理的领地

你可以：
- 改领地名字，比如从"小破屋"改成"钻石宫殿"
- 设置欢迎语，比如"欢迎来到我的钻石屋！"、"熊孩子勿入"、"这里有苦力怕"
- 添加成员，让朋友来帮忙建房子
- 设置权限，决定谁能挖东西、谁能放东西
- 删除领地，不想要了就删

### 权限是什么？

简单来说就是「谁能在我地盘干什么」：

| 权限 | 说明 | 默认 |
|------|------|------|
| 破坏方块 | 能不能挖东西 | ❌ 不能 |
| 放置方块 | 能不能放东西 | ❌ 不能 |
| 与方块交互 | 能不能开箱子 | ❌ 不能 |
| 与实体交互 | 能不能和村民交易 | ❌ 不能 |

### 谁有什么权限？

| 人物 | 说明 |
|------|------|
| 管理员 | 服务器 OP，啥都能干 |
| 领地主 | 你自己，管理你的领地 |
| 领地成员 | 你添加的朋友 |
| 群组成员 | 你群组里的人 |
| 访客 | 其他所有人，默认啥都不能干（熊孩子除外） |

### 开放群组

你可以设置领地对某些群组开放，群组成员会自动获得「群组成员」权限，不用一个个添加。

## 常见问题

### 为什么创建不了领地？

- 检查管理员有没有开启领地功能
- 确认你选了两个点（一个角和对角）
- 检查你有没有足够的游戏币
- 确认记分板配置正确

### 为什么别人还能挖东西？

- 确认你创建的是「三维领地」（有高度限制）
- 检查访客权限设置
- 确认对方不是管理员

### 我能创建几个领地？

默认每人最多 2 个领地，管理员可以调整。

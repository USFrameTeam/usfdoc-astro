策略文件可以让你创建自己的菜单界面，就像服务器商店那样的！

## 什么是策略文件？

策略文件就是一个「自定义菜单」，你可以：
- 设计菜单界面
- 添加按钮
- 按钮可以执行命令、打开商店、传送到某个地方

## 怎么获取策略文件？

- 创造模式物品栏里找「策略文件」
- 或者输入命令：`/give @s usf:config_file`

<Aside type="note">
策略文件有附魔光泽，看起来亮闪闪的！
</Aside>

## 怎么编辑策略文件？

<Steps>

1. 手持策略文件

2. 输入 `+op` 打开管理界面

3. 点击「编辑策略文件」

4. 开始设计你的菜单！

</Steps>

## 怎么使用策略文件？

### 直接打开
手持策略文件，对着空气右键，菜单就会弹出来！

### 绑定到方块
手持策略文件，按住 Shift（潜行），右键一个方块。

这样别人右键那个方块就能打开菜单了！

### 绑定到NPC
手持策略文件，右键一个NPC。

这样别人右键那个NPC就能打开菜单了！

## 按钮可以做什么？

| 按钮类型 | 说明 |
|----------|------|
| 执行命令 | 点击后自动执行一个命令 |
| 打开商店 | 跳转到商店的某个商品 |
| 传送到某个地方 | 直接传送到指定位置 |
| 打开子菜单 | 再打开一个策略文件界面 |

## 我能做什么样的菜单？

你可以做：
- 服务器商店
- 传送菜单
- 帮助信息
- 任务系统
- 任何你能想到的！

策略文件可以让你创建自己的菜单界面，就像服务器商店那样的！

## 什么是策略文件？

策略文件就是一个「自定义菜单」，你可以：
- 设计菜单界面
- 添加按钮
- 按钮可以执行命令、打开商店、传送到某个地方

## 服主看这里

### 获取策略文件

- 创造模式物品栏里找「策略文件」
- 或者输入命令：`/give @s usf:config_file`

<Aside type="note">
策略文件有附魔光泽，看起来亮闪闪的！
</Aside>

### 编辑策略文件

<Steps>

1. 手持策略文件（放在副手）

2. 输入 `+op` 打开管理界面

3. 点击「编辑策略文件」

4. 开始设计你的菜单！

</Steps>

### 编辑内容

- **修改配置**：设置标题、内容、关闭操作等
- **二级菜单**：创建子菜单
- **添加按钮**：添加各种功能按钮
- **删除按钮**：删除不需要的按钮
- **预览**：看看效果

### 绑定到方块/NPC

- **绑定方块**：手持策略文件，按住 Shift（潜行），右键方块
- **绑定NPC**：手持策略文件，右键NPC

这样玩家右键那个方块/NPC就能打开菜单了！

### 按钮类型

| 按钮类型 | 说明 | 用途 |
|----------|------|------|
| 执行命令 | 点击后执行命令 | 传送、领取奖励等 |
| 打开商店 | 跳转到商店 | 快速购物 |
| 传送点 | 传送到指定位置 | 快速移动 |
| 子菜单 | 打开另一个界面 | 多级菜单 |

### 能做什么？

你可以做：
- 服务器商店（比原版商店更灵活）
- 传送菜单（一键到达常用地点）
- 帮助信息（新手指引）
- 任务系统（领取任务）
- 任何你能想到的！

## 玩家看这里

### 打开菜单

- 右键绑定的方块/NPC
- 或者手持策略文件，对着空气右键

### 菜单能做什么？

看服主怎么设计了，可能是：
- 买东西
- 传送到某个地方
- 领取奖励
- 查看信息
- 等等...

<Aside type="tip">
如果菜单有二级菜单，点击按钮可以进入子菜单，返回按钮可以回到主菜单。
</Aside>

策略文件（`usf:config_file`）是 USF 的自定义 UI 系统，可以创建交互式菜单界面。

## 获取策略文件

- 创造模式物品栏中搜索「策略文件」
- 使用命令 `/give @s usf:config_file`

<Aside type="note">
策略文件具有附魔光泽，不可堆叠。
</Aside>

## 编辑策略文件

<Steps>

1. 手持策略文件

2. 输入 `+op` 打开管理界面

3. 管理界面 → 编辑策略文件

4. 设置界面内容：
   - 主界面标题
   - 内容说明
   - 二级菜单
   - 按钮配置

</Steps>

## 按钮类型

| 类型 | 说明 |
|------|------|
| 执行命令 | 点击后执行指定命令 |
| 打开商店 | 跳转到商店指定商品 |
| 传送点 | 传送到指定位置 |
| 子页面 | 打开另一个策略文件界面 |

## 使用策略文件

### 右键空气

手持策略文件，右键空气打开配置的界面。

### 绑定到方块

手持策略文件，潜行+右键方块，将界面绑定到该方块。

其他玩家右键该方块时会打开界面。

### 绑定到NPC

手持策略文件，右键NPC，将界面绑定到该NPC。

其他玩家右键该NPC时会打开界面。

## 配置文件结构

```javascript
{
  title: "",      // 界面标题
  body: "",       // 界面说明
  groups: {},     // 二级菜单
  things: []      // 按钮列表
}
```

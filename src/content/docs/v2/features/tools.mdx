## 小木斧

小木斧是一个快速建造工具，可以帮你快速填充方块、导出建筑。

### 服主指南

管理界面 → 插件设置 → 其他功能 → 小木斧

开启后，手持木斧的OP可以使用小木斧功能。

### 使用方法

<Steps>

1. **获取木斧**：在创造模式拿一把木斧

2. **选择第一个点**：手持木斧，左键点击一个方块
   - 会看到粒子效果

3. **选择第二个点**：右键点击对角位置的方块
   - 会看到一个框框

4. **打开操作面板**：按住 Shift（潜行），右键空气

5. **选择操作**：
   - 填充方块（全部替换）
   - 填充方块（只替换空气）
   - 填充方块（只做轮廓）
   - 导出为结构文件

</Steps>

<Aside type="caution">
小木斧需要管理员在配置中开启才能使用！
</Aside>

### 能做什么？

- **快速建造**：瞬间填满一片区域，盖房子效率翻倍
- **快速清除**：瞬间清空一片区域，拆建筑不费劲
- **导出建筑**：把你的建筑保存为结构文件，可以复制粘贴

## 悬浮字

悬浮字是一种悬浮在空中的文字，可以用来做公告、路标等。

### 服主指南

管理界面 → 管理界面 → 管理悬浮字

<Steps>

1. 点击「添加悬浮字」

2. 设置文字内容

3. 设置位置（当前坐标或手动输入）

</Steps>

### 能显示什么？

除了普通文字，还可以用变量显示动态信息：
- 当前时间
- 在线玩家数量
- 世界运行时间
- 记分板分数
- 等等...

### 特点

- 悬浮在空中，不会掉下来
- 不会自然消失
- 可以随时编辑或删除
- 支持变量转义

## 双击告示牌

在 USF 里，你可以双击告示牌来编辑内容，比原版方便多了！

<Aside type="tip">
管理员可以开启「告示牌更改记录」，谁改了告示牌一目了然！
</Aside>

## 记分板管理

管理界面 → 管理界面 → 记分板管理

服主可以：
- 创建/删除记分板
- 设置显示位置（侧边栏/玩家列表）
- 修改玩家分数
- 设置默认值

<Aside type="note">
记分板是USF很多功能的基础，比如领地费用、商店货币等。
</Aside>

## 背包检查

管理界面 → 玩家管理 → 背包检查

管理员可以查看其他玩家的背包内容，方便查证物品丢失等问题。

<Aside type="tip">
背包检查会生成一个潜影盒物品，里面包含目标玩家的背包内容。
</Aside>

## 跟踪视角

管理界面 → 玩家管理 → 跟踪视角

管理员可以跟踪其他玩家的视角，方便观察玩家行为。

跟踪模式：
- **第一视角**：以目标玩家的视角观察
- **自由视角**：在目标玩家附近自由观察

<Aside type="note">
跟踪期间输入 `+op` 可退出跟踪。
</Aside>

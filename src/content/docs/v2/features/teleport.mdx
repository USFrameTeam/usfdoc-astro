传送系统让玩家在服务器里快速移动，无论是互传还是设置一个传送点。

## 服主指南

### 功能配置

管理界面 → 插件设置 → 传送设置

| 功能 | 说明 | 建议 |
|------|------|------|
| 个人传送点 | 玩家自定义传送点 | 推荐开启 |
| 传送点上限 | 每位玩家可创建的传送点数量 | 默认50，满足日常需求 |
| 玩家互传 | TPA功能 | 推荐开启，便于社交互动 |
| 返回死亡点 | 死亡后返回遗骸位置 | 新手服务器建议开启 |
| 冷却时间 | 传送间隔限制 | 30秒为宜，避免滥用 |
| 随机传送 | 随机位置传送 | 根据地图规模调整范围 |
| 传送动画 | 摄像机传送效果 | 可选，提升沉浸感 |

### 公共传送点

管理界面 → 传送设置 → 世界公共点

设置常用地标（出生点、商业区、活动区等），方便玩家快速到达核心区域。

## 玩家指南

### 我 想 回 家！

别急，按照下面流程，设定一个有关于你的个人传送点，可以快速的传送回指定地点!

<Steps>

1. 主菜单 → 传送系统 → 个人传送点 → 添加传送点

2. 给传送点取个名字，比如"我的家"、"111"。

3. 勾选「设为Home」

4. 以后想回家，直接输入 `+home` 就行了！

</Steps>

<Aside type="tip">
你最多可以创建 50 个传送点，但只能有 1 个 Home 点！别贪心~
</Aside>

### 我要找朋友！

传送其他人到你这里，或者传送你到其他人的所在点位

<Steps>

1. 主菜单 → 传送系统 → 传送玩家

2. 选择你想找的朋友

3. 选择「我过去」或者「他过来」

4. 朋友输入 `+tpaccept` 同意

</Steps>

<Aside type="note">
传送有冷却时间（默认30秒），不能当瞬移机器用！
</Aside>

### 我要探险！

输入 `+tpr` 随机传送，去一个完全陌生的地方，开盲盒式探险！

运气好可能传到合适的据点，运气不好可能传到神秘古城，谨慎使用！

### 我死了想回去捡装备！

输入 `+back` 回到死亡地点.

<Aside type="danger">
如果管理员没开这个功能，你就只能跑回去了，哭~
</Aside>

### 我想分享传送点给朋友

你发现了一个钻石矿，想让朋友也来挖？传送系统支持分享传送点！

<Steps>

1. 选择一个传送点

2. 点击「分享」按钮

3. 设置分享时间（10秒 ~ 10分钟）

4. 选择分享给「所有玩家」还是「部分玩家」

</Steps>

<Aside type="tip">
分享的传送点是临时的，时间到了就没了，抓紧时间！
</Aside>

## 传送点类型

### 个人传送点

- 新建传送点可以选择当前位置，也可以从个人传送点复制过来
- 每个玩家最多可创建50个个人传送点
- 可以设置一个Home点，使用 `+home` 快速传送

### 固定传送点

- 固定传送点只能由管理员设置
- 直接显示传送点在传送点主界面

### 世界公共点

- 世界公共点则是全服可添加
- 可以通过全局事件对玩家使用传送点进行响应

### 管理玩家个人点

插件OP可以在个人传送点界面点击“管理玩家个人点”，来查看某玩家的全部个人点。

## 传送动画

管理员可以开启「传送动画」，传送时会有一个模仿古人的摄像机效果：
- 镜头升起
- 移动到目标位置上方
- 缓缓降落

不开动画的话就是瞬间传送，看你喜欢哪种风格！

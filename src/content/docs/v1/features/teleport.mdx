## 传送点系统

V1支持多种传送点类型，满足不同场景需求。

### 个人传送点

每个玩家可创建多个个人传送点。

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| pos | 常用传送点数量 | 10 |
| more | 更多传送点数量 | 40 |

**使用方式**：主菜单 → 传送 → 个人传送点

### 世界共享点

所有玩家共享的传送点，由管理员设置。

| 配置项 | 说明 |
|--------|------|
| world | 世界共享点功能开关 |

### 出生点传送

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| spawn | 返回出生点功能开关 | 开启 |
| y | 出生点高度(y坐标) | 64 |

### 随机传送

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| random | 随机传送范围 | 50000 |

设置为0则不显示随机传送按钮。

## TPA传送

玩家之间可互相传送请求。

### 功能配置

| 配置项 | 说明 |
|--------|------|
| tpa | TPA功能开关 |

### 使用流程

<Steps>

1. 玩家A发送TPA请求
2. 玩家B收到请求提示
3. 玩家B输入 `tpaccept` 接受请求
4. 玩家A传送到玩家B位置

</Steps>

## 死亡传送

| 配置项 | 说明 |
|--------|------|
| die | 返回死亡坐标功能开关 |

玩家死亡后可通过菜单返回死亡位置。

## 跨维度传送

| 配置项 | 说明 |
|--------|------|
| DiEffect | 跨维度时给予抗性效果 |

开启后，玩家跨维度传送时会获得10秒抗性提升III效果，防止落地伤害。

## 传送点配置

传送点系统支持按组别(Group)配置不同界面：

```json
{
  "pos": {
    "y": 64,
    "all": ["tpa", "die", "world", "spawn"],
    "pos": 10,
    "random": 50000,
    "more": 40
  }
}
```

### 配置说明

| 字段 | 类型 | 说明 |
|------|------|------|
| y | 数字 | 出生点高度 |
| all | 数组 | 启用的传送功能列表 |
| pos | 数字 | 常用传送点数量上限 |
| random | 数字 | 随机传送范围 |
| more | 数字 | 更多传送点数量上限 |

## 可用传送功能

| 功能ID | 说明 |
|--------|------|
| tpa | TPA玩家互传 |
| die | 返回死亡坐标 |
| world | 世界共享点 |
| spawn | 返回出生点 |

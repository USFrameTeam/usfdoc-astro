## 聊天格式

V1支持自定义聊天格式，可使用变量替换。

### 默认格式

```
[/tag]§b/sender§r >> /text
```

### 可用变量

| 变量 | 说明 |
|------|------|
| /sender | 发送者名称 |
| /tag | 聊天头衔 |
| /text | 聊天内容 |

### 聊天头衔

可通过scriptevent设置玩家聊天头衔：

```
/scriptevent usf:tag 玩家名 头衔内容
```

## 私聊系统

玩家之间可进行私聊。

### 命令

| 命令 | 说明 |
|------|------|
| talk | 打开私聊界面 |
| 私聊 | 打开私聊界面 |

### 功能

- 选择在线玩家进行私聊
- 私聊消息仅双方可见
- 支持Emoji表情

## 聊天禁言

系统支持临时禁言功能，通过设置玩家的 `enable_chat` 数据实现。

### 禁言状态查询

禁言剩余时间可通过代码获取：

```javascript
chat_left_time(player) // 返回剩余毫秒数
```

### 禁言判断

```javascript
can_chat(player) // 返回true表示可以发言
```

## 天气提示

## 留言板

玩家可在告示牌上创建留言板。

### 创建方式

运行命令：
```
/scriptevent usf:sign_board
```

### 功能特性

- 玩家可留言
- 支持清空留言
- 可关闭留言板
- 留言板实体使用潜影弹实现

### 界面文本

| 文本ID | 内容 |
|--------|------|
| sign_board/title | 留言板 |
| sign_board/nothing | 这里没有留言 |
| sign_board/add | 留言 |
| sign_board/clear | 清空 |
| sign_board/kill | 关闭此留言板 |

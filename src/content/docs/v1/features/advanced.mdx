## 翻译系统

开启 `Tran` 功能后，支持文本变量替换。

### 可用变量

#### 通用变量

| 变量 | 说明 |
|------|------|
| /pos | 玩家当前位置 |
| /list | 玩家列表 |
| /name | 玩家名称 |
| /alltime | 世界运行总时间(秒) |
| /board.计分板ID.score | 玩家计分板分数 |
| /board.计分板ID.name | 该计分板显示名称 |
| /dimension | 玩家所在维度 |

#### 聊天专用变量

| 变量 | 说明 |
|------|------|
| /sender | 发送者名称 |
| /tag | 聊天头衔 |
| /text | 聊天内容 |

## 日志系统

开启 `Log` 功能后，记录关键操作日志。

### 记录内容

- 玩家破坏危险方块
- 玩家放置危险方块
- 玩家获得违禁物品
- 附魔等级异常
- 范围挖掘检测

### 日志格式

```
[时间] 玩家名 坐标 (维度) 操作描述
```

## 数据实体

V1使用数据实体存储大量数据。

### 数据实体创建

- 使用潜影弹(Shulker Bullet)作为数据实体
- 生成在坐标 (0, -64.5, 0)
- 使用实体事件触发数据存储模式

### 数据实体用途

| 用途 | 说明 |
|------|------|
| 队伍数据 | 存储队伍信息 |
| 页面内容 | 存储自定义页面 |
| 交易数据 | 存储交易信息 |

### 数据实体保护

- 位于基岩层下方
- 每100tick传送到安全位置
- 自动检测并重新创建

## 多语言支持

V1支持中英双语言切换。

### 语言标识

文本使用 `§?` 分隔中英文：

```
中文§?English
```

### 语言切换

玩家可通过标签设置语言：

```json
{"lang": 0}  // 中文
{"lang": 1}  // 英文
```

## Scriptevent命令

V1通过scriptevent系统接收命令。

### 命令格式

```
/scriptevent usf:命令ID 参数
```

### 可用命令

| 命令ID | 参数 | 说明 |
|--------|------|------|
| manager | 无 | 打开管理界面 |
| reset | 无 | 重置最高OP |
| unban | 玩家名 | 解封玩家 |
| sign_board | 无 | 创建留言板 |
| tag | 玩家名,头衔 | 设置聊天头衔 |
| open | 页面ID | 打开指定页面 |
| owner | 无 | 查看最高OP |
| group | add/remove/list | 管理玩家组 |

### 错误提示

命令不存在时返回：
```
[USF]你所找的命令"X"不存在
```

## 界面配置

### 界面编辑

支持按组别(Group)配置不同界面。

#### 配置结构

```json
{
  "menu": [
    {
      "group": "*",
      "all": ["pos", "player", "me", "chat", "board", "about", "kill"]
    },
    {
      "group": "vip",
      "all": ["pos", "player", "me", "chat"]
    }
  ]
}
```

#### 可配置界面

| 界面ID | 说明 |
|--------|------|
| pos | 传送界面 |
| player | 玩家互动界面 |
| me | 我的界面 |
| chat | 聊天设置界面 |
| board | 公告界面 |
| about | 关于界面 |
| kill | 自杀按钮 |

### 配置管理

- 支持添加/删除配置
- 支持上下移动优先级
- 支持复制配置
- 优先匹配第一个符合条件的配置

## 自定义菜单物品

### 配置

```json
{
  "menu_item": ["minecraft:clock"]
}
```

玩家持有指定物品时可打开主菜单。

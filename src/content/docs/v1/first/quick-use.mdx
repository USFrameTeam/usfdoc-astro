## 前置环境

| 项目 | 需求 |
|---|---|
| Minecraft版本 | Minecraft基岩版1.19.50 - 1.20.60 |
| 实验性选项 | 测试版API |
| 服务端类型 | BDS/llse/Endstone或Minecraft游戏 |

## 安装方法

1. 下载V1版本的mcaddon文件
2. 新建存档/打开存档设置
3. 打开实验性玩法「测试版API」
4. 安装USF行为包
5. 进入游戏
6. 按照提示输入 `/function get_owner` 命令获取插件Owner
